é um problema no React que ocorre quando você precisa passar dados por vários níveis de componentes que não utilizam esses dados diretamente, mas apenas os transmitem para outros componentes mais profundos na árvore. Esse processo pode complicar o código e torná-lo difícil de manter, especialmente em aplicações grandes, E com o Context API permite que você compartilhe dados entre componentes sem precisar passar props manualmente por cada nível da árvore de componentes.
